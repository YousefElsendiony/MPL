# MPL
COMS W4115 Spring 2019 Project

Currently inherited alot of functionality from Stephen Edwards microC compiler. 

# Installation Steps
$ sudo apt install ocaml llvm llvm-runtime m4 opam

$ opam init

$ opam install llvm.3.8

$ eval \`opam config env\`


# Test Run

------------------------------
To run and test:

$ make

$ ./testall.sh
